jedeschule.codefor.de

jedeschule.codefor.de bringt Stammdaten aller Schulen Deutschlands an einem Ort zusammen

  • per API
  • als CSV-Datei zum Download
  • zum interaktiven Erkunden

Geschichte

Ursprünglich als jedeschule.de gestartet, brachte die Open Knowledge Foundation 2017 in Zusammenarbeit mit BildungsCent erstmals eine Rechercheplattform ans Licht der Welt, auf der Stammdaten von Schulen (Name, Standort, Schulform, Träger) erstmals für ganz Deutschland verfügbar waren.
Für einige Bundesländer wurden außerdem Informationen zu Partnerschaften und Aktivitäten außerhalb des Unterrichts bereitgestellt.

Gesammelt wurden die Daten per Webscraping. Im Laufe der Zeit änderten die Bundesländer die Struktur ihrer Webseiten und die Scraper konnten zunehmend keine neuen Daten mehr abrufen.

Seit 2019 kümmern sich Freiwillige von codefor.de um die Aktualisierung der Scraper und Daten. Seitdem werden einmal pro Woche die neusten Daten abgerufen und kontinuierlich bereitgestellt.

Daten

Alle Daten zum Projekt sind unter Daten frei zugänglich und können heruntergeladen werden. Haben Sie Anmerkungen oder einen Fehler entdeckt, dann melden Sie sich gerne bei uns per Mail.

Die aktuellen Daten werden von folgenden Quellen bezogen:

  • Baden-Württemberg: Schul- und Dienststellensuche
  • Berlin: Geodatenportal (FISBroker)
  • Brandenburg: Geodatenportal
  • Bremen: Schulwegweiser
  • Hamburg: Geodatenportal
  • Hessen: Schuldatenbank
  • Mecklenburg-Vorpommern: Verzeichnisse zum Download (Datenstand 2023, weil aktuelle Daten nur zum Kauf angeboten werden)
  • Niedersachsen: Bildungsportal - Schulsuche
  • Nordrhein-Westfalen: Open Data Portal des Schulministeriums
  • Rheinland-Pfalz: Bildungsserver - Schulsuche
  • Saarland: Bildungsserver - Schulsuche
  • Sachsen: Schuldatenbank
  • Sachsen-Anhalt: Schulsuche
  • Schleswig-Holstein: Open Data Portal
  • Thüringen: Schulportal
